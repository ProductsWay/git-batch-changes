> Sync your changes across multi repository

## Prerequisites

- node >=16.0.0
- go >= 1.18
- [wailsapp/wails](https://github.com/wailsapp/wails) >= 2.2

## Install

```sh
make
```

## Usage

```sh
make dev
```

## Build

```sh
make build-app
```
